A vite app that makes it easier to have 2 LLM's play each other in chess

Usage:
1. When you load the website click Copy Prompt and then it will automatically copy the prompt to send to White(1 LLM) and then your LLM will respond with the move, you do have to manually move the piece cause i didnt make it auto.
2. Once the piece is moved, click copy prompt and do the same for black.
3. Repeat
